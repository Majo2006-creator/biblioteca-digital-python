# Sistema de Gestión de Biblioteca Digital

Proyecto en Python que simula un sistema de gestión de biblioteca.

## Funcionalidades

- Añadir libros
- Eliminar libros
- Registrar usuarios
- Prestar libros
- Devolver libros
- Buscar libros por:
  - título
  - autor
  - categoría

## Estructuras utilizadas

- Tuplas → datos inmutables del libro
- Listas → libros prestados a usuarios
- Diccionarios → almacenamiento de libros por ISBN
- Conjuntos → control de IDs únicos

## Ejecución

```bash
python biblioteca.py
